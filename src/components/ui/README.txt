COMPONENTES UI - Componentes básicos reutilizables

Aquí van componentes primitivos como:

Button/ - Botones (primario, secundario, terciario)
Input/ - Campos de formulario
Card/ - Tarjetas contenedoras
Badge/ - Etiquetas (nuevo, oferta, agotado)
Modal/ - Ventanas modales/dialogs
Dropdown/ - Menús desplegables
Spinner/ - Indicadores de carga
Toast/ - Notificaciones emergentes

Cada componente debe tener:
- ComponentName.jsx
- ComponentName.module.css
