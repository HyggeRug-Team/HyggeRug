COMPONENTES DE RESEÑAS - Sistema de opiniones

ReviewCard/ - Tarjeta de reseña individual
  - Nombre del usuario
  - Rating (estrellas)
  - Título de la reseña
  - Comentario
  - Fecha
  - Badge "Compra verificada" (si aplica)

ReviewForm/ - Formulario para dejar reseña
  - Selector de estrellas (1-5)
  - Campo título
  - Campo comentario
  - Botón "Publicar reseña"
  - Solo si ha comprado el producto

StarRating/ - Componente de estrellas
  - Versión solo lectura (mostrar rating)
  - Versión interactiva (seleccionar rating)
  - Número de estrellas llenas/vacías

ReviewList/ - Lista de todas las reseñas
  - Ordenar por (más recientes, mejor valoradas)
  - Filtrar por rating
  - Paginación
