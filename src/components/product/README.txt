COMPONENTES DE PRODUCTO - Todo lo relacionado con productos

ProductCard/ - Tarjeta de producto
  - Imagen
  - Nombre
  - Precio
  - Badge (nuevo, oferta)
  - Botón "Ver más"
  - Hover effects

ProductGrid/ - Grid responsivo de productos
  - Layout en grid
  - Responsive (móvil/tablet/desktop)

ProductDetail/ - Vista detallada del producto
  - Toda la información completa

ProductGallery/ - Galería de imágenes
  - Imagen principal
  - Thumbnails laterales
  - Zoom al hacer click

ProductInfo/ - Info del producto
  - Nombre
  - Precio
  - Descripción
  - Dimensiones
  - Material
  - Colores
  - Stock disponible

ProductFilters/ - Filtros laterales
  - Por categoría
  - Por precio
  - Por tamaño
  - Por colores

SizeSelector/ - Selector de tamaño
ColorPalette/ - Selector de colores
RelatedProducts/ - Productos relacionados/similares
