COMPONENTES DE ADMIN - Panel de administración

ProductForm/ - Formulario de producto (crear/editar)
  - Nombre
  - Descripción
  - Precio
  - Precio con descuento
  - Categoría
  - Dimensiones (ancho/alto)
  - Material
  - Técnica
  - Colores
  - Stock
  - Imágenes (subida múltiple)
  - Destacado (sí/no)
  - Activo (sí/no)

OrderManagement/ - Gestión de pedidos
  - Lista de pedidos
  - Cambiar estados
  - Añadir tracking
  - Ver detalles

StatsCard/ - Tarjeta de estadísticas
  - Título
  - Valor numérico
  - Icono
  - Tendencia (subida/bajada)
  - Usado en dashboard

DataTable/ - Tabla de datos reutilizable
  - Cabeceras
  - Filas de datos
  - Ordenamiento
  - Paginación
  - Acciones (editar/eliminar)
