COMPONENTES COMUNES - Utilities y helpers

SearchBar/ - Barra de búsqueda
  - Input de búsqueda
  - Icono de lupa
  - Botón buscar
  - Sugerencias/autocompletado (opcional)

Breadcrumbs/ - Migas de pan
  - Home > Categoría > Producto
  - Links navegables
  - Último item sin link

Pagination/ - Componente de paginación
  - Botones anterior/siguiente
  - Números de página
  - Página actual destacada
  - Ir a primera/última

ErrorBoundary/ - Captura de errores React
  - Captura errores de componentes hijos
  - Muestra UI de error amigable
  - Opción de recargar

LoadingState/ - Estado de carga
  - Skeleton loaders
  - Spinners
  - Mensajes de carga
