COMPONENTES DE CARRITO - Carrito de compras

CartItem/ - Item individual del carrito
  - Imagen del producto
  - Nombre
  - Precio
  - Selector de cantidad (+/-)
  - Botón eliminar
  - Subtotal

CartSummary/ - Resumen del carrito
  - Subtotal
  - Envío
  - Impuestos
  - Total
  - Botón "Proceder al pago"

CartSidebar/ - Sidebar del carrito (deslizante)
  - Lista de items
  - Resumen
  - Se abre/cierra con animación

EmptyCart/ - Estado cuando el carrito está vacío
  - Icono
  - Mensaje
  - Botón "Ir a la tienda"
