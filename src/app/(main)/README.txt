PÁGINAS - LAYOUT PRINCIPAL (main)

Este grupo de rutas usa el layout principal con Navbar y Footer.

page.jsx - Homepage / Landing page principal
  - Hero section
  - Productos destacados
  - Categorías
  - Testimonios
  - Newsletter

tienda/page.jsx - Catálogo completo de alfombras
  - Grid de productos
  - Filtros laterales
  - Paginación
  - Búsqueda

tienda/[slug]/page.jsx - Detalle de un producto
  - Galería de imágenes
  - Información del producto
  - Precio y stock
  - Botón añadir al carrito
  - Productos relacionados
  - Reseñas

categorias/[slug]/page.jsx - Productos por categoría
  - Filtrados por categoría específica
  - Mismo layout que tienda

sobre-nosotros/page.jsx - Sobre Hygge Rug
  - Historia de la marca
  - Proceso de tufting
  - Equipo

contacto/page.jsx - Formulario de contacto
  - Email
  - Teléfono
  - Redes sociales
  - Mapa (opcional)

layout.jsx - Layout con Navbar + Footer
  - Navbar en la parte superior
  - {children} en el medio
  - Footer en la parte inferior
